# Changelog

All notable changes to the Glass Empires SRD.

## 1.0.1 — 2026-05-19

### Changed
- Replaced moves content collection with actions — all moves migrated to action entries
- Restructured magic content: consolidated magic rules, removed redundant move-casting/move-spell-dice files
- Removed standalone glossary pages — terms now defined inline in relevant sections
- Removed redundant faction move pages (all now handled as actions)
- Enhanced offline ZIP packaging: asset inlining, search guard, startup scripts, depth-aware path rewriting
- Updated nav.ts to match restructured content hierarchy

### Added
- 404 page with search suggestions via Pagefind
- Page preview hover tooltips on internal links
- Faction actions page documenting faction-level actions
- GM inspirations page
- Spellcasting, spell dice & totems, and what-magic-is pages

## 1.0.0 — 2026-05-15

### New
- Initial public release of the Astro-based SRD site
- Downloadable offline version
- Versioned releases with update notifications
